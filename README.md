> auralization grep command

## Installation

```
$ npm install -g mimi-grep
```

## Usage

```
$ mimi-grep filename
```

```
$ mimi-grep -c beep filename
```

```
$ mimi-grep -c drone filename
```

```
$ mimi-grep -c drum filename
```
